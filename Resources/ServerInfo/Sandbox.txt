
[color=#ffffff]Опції панелі створення сутностей[/color]
[color=#a4885c]За замовчуванням[/color] створює невеликі сутності, такі як кухлі, не вирівнюючи їх ні до чого, тоді як блокові сутності, такі як стіни, вирівнюються по сітці.
[color=#a4885c]Вільне розміщення[/color] створює всі сутності без вирівнювання.
[color=#a4885c]Розмістити поруч[/color] обмежує радіус створення приблизно до 2 плиток.
[color=#a4885c]Прив'язати до центру сітки[/color] вирівнює сутність по центру плитки.
[color=#a4885c]Прив'язати до межі сітки[/color] вирівнює сутність по межі плитки.

[color=#ffffff]Опції вирівнювання по сітці[/color]
[color=#a4885c]Вирівняти за подібними[/color] прив'язує сутність до подібних сутностей. Наразі не працює.
[color=#a4885c]Вирівняти по будь-якій плитці[/color] вирівнює сутність по сітці.
[color=#a4885c]Вирівняти по порожній плитці[/color] цільова плитка повинна бути порожньою
[color=#a4885c]Вирівняти по нещільній плитці[/color] у цільовій плитці не допускаються колайдери.
[color=#a4885c]Вирівняти по щільній плитці[/color] у цільовій плитці повинні бути колайдери.
[color=#a4885c]Вирівняти по стіні[/color] прив'язується до вертикальних напівплиток.
[color=#a4885c]Правильне вирівнювання по стіні[/color] прив'язує сутність до середини країв плитки
